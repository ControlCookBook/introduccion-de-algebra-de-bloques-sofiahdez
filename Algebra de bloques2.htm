Algebra de bloques

 

Elementos de un diagrama de bloques 

Bloques: es el elemento principal de un diagrama de bloques ya que representa los componentes del sistema. Es el elemento que contiene la relación entre la entrada y la salida del componente del sistema, ya sea con un nombre o directamente la función de transferencia del elemento.

 

 

Líneas: son las que representan la trayectoria de las señales y su sentido. Indican la conexión de una variable entre diversos bloques.

Punto de suma: representado por un círculo que indica que la salida es igual a la suma de las dos señales que entran.

 

 

 

 

Los diagramas de bloques del sistema de control se pueden simplificar usando una serie de teoremas de transformación mostrados a continuación.

 

 

 

Ejemplos donde podemos utilizar los teoremas:

·         Simplificación de un sistema de bloques complejos.

·         Diagrama de bloques de varias entradas.

·         Entradas y salidas múltiples.

·         Construcción de un diagrama de bloques a partir de las funciones de transferencia de los elementos del sistema.
